# Bank Operation

## Objective
Creating a basic bank operations (deposit, withdraw, check balance, and transaction history) using Java Object-Oriented Programming.

---

## Tools Used
- Java  
- VS Code  
- Terminal

---

## How to Run
1. Open terminal and run below 2 in order to run
   To Compile: javac BankApp.java and javac Account.java
   To Run: java BankApp and java Account

---

## Features: 
- Create account with initial balance
- Deposit and withdraw money
- Check account balance
- View transaction history
